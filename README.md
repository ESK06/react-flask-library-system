# Library Managment System

A Library Managment System designed using **React**, **Flask** and **MongoDB**.
This project includes both **automated testing with Playwright** and **API testing using Postman**, along with manual test documentation.
Also includes **Docker** configuration for Deployment.

## Technologies
- **Frontend:** React
- **Backend:** Flask
- **Database:** MongoDB
- **Testing:** Playwright, Postman
- **Deployment:** Docker

## Project Structure
```
/tests
  Playwright Tests
/QA
  Manual Testing.xlsx - Manual test cases
/QA/postman
  Library Flask API Collection and Flask Environment Json Files
```
